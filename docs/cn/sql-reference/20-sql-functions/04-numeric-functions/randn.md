---
title: RAND(n)
---

返回一个在范围 0 <= v < 1.0 内的随机浮点数值 v。要获得一个在范围 i <= R < j 内的随机整数 R，使用表达式 FLOOR(i + RAND() * (j − i))。参数 `n` 被用作种子值。对于相等的参数值，RAND(n) 每次返回相同的值，因此产生一列值的可重复序列。

## 语法

```sql
RAND( <n>)
```

## 示例

```sql
SELECT RAND(1);

┌────────────────────┐
│       rand(1)      │
├────────────────────┤
│ 0.7133693869548766 │
└────────────────────┘
```
