---
title: BLAKE3
---

计算字符串的 BLAKE3 256位校验和。如果参数为 NULL，则返回值为 64 位十六进制数字的字符串，或者为 NULL。

## 语法 {/*examples*/}

```sql
BLAKE3(<expr>)
```

## 示例 {/*examples*/}

```sql
SELECT BLAKE3('1234567890');

┌──────────────────────────────────────────────────────────────────┐
│                       blake3('1234567890')                       │
├──────────────────────────────────────────────────────────────────┤
│ d12e417e04494572b561ba2c12c3d7f9e5107c4747e27b9a8a54f8480c63e841 │
└──────────────────────────────────────────────────────────────────┘
```
