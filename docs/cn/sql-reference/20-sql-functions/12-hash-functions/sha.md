---
title: SHA
---

计算字符串的 SHA-1 160位校验和，如 RFC 3174 (安全哈希算法) 所描述。返回值为一个40位十六进制数字的字符串，如果参数为 NULL 则返回 NULL。

## 语法

```sql
SHA(<expr>)
```

## 示例

```sql
SELECT SHA('1234567890');

┌──────────────────────────────────────────┐
│             sha('1234567890')            │
├──────────────────────────────────────────┤
│ 01b307acba4f54f55aafc33bb06bbbf6ca803e9a │
└──────────────────────────────────────────┘
```
