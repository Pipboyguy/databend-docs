---
title: ARRAY_SORT 数组排序
---

按升序排序数组中的元素。

## 语法 {/*syntax*/}

```sql
ARRAY_SORT( <array>[, <order>, <nullposition>] )
```

| 参数          | 默认值       | 描述                                                                                               |
|--------------|-------------|----------------------------------------------------------------------------------------------------|
| order        | ASC         | 指定排序顺序，可以是升序（ASC）或降序（DESC）。                                                    |
| nullposition | NULLS FIRST | 确定排序结果中 NULL 值的位置，在开头（NULLS FIRST）或结尾（NULLS LAST）的排序输出中。             |

## 示例 {/*examples*/}

```sql
SELECT ARRAY_SORT([1, 4, 3, 2]);

┌──────────────────────────┐
│ array_sort([1, 4, 3, 2]) │
├──────────────────────────┤
│ [1,2,3,4]                │
└──────────────────────────┘
```
