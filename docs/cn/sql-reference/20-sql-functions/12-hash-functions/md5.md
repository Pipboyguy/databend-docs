---
title: MD5
---

计算字符串的 MD5 128位校验和。如果参数为 NULL，则返回值为32位十六进制数字的字符串，或者为 NULL。

## 语法 {/*syntax*/}

```sql
MD5(<expr>)
```

## 示例 {/*examples*/}

```sql
SELECT MD5('1234567890');

┌──────────────────────────────────┐
│         md5('1234567890')        │
├──────────────────────────────────┤
│ e807f1fcf82d132f9bb018ca6738a19f │
└──────────────────────────────────┘
```
