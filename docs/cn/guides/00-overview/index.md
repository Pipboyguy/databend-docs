# Databend 指南

欢迎来到 Databend 指南，这里涵盖了 [Databend 版本](00-editions/index.md)的所有方面。
文档中清楚地标注了特定版本独有的功能。

**`Databend`** 指的是开源版本，而 **`Databend Cloud`** 是一个完全托管的云服务。**`Databend(Cloud)`** 统称 Databend 和 Databend Cloud。
